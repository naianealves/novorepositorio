Fazendo uma alteração, para ser feito um novo comit e u um novo versionamento

Um branch é uma ramificação

Um merge é uma fundição da ramificaçã branch com o código principal

Um commit significa o arquivo alterado salvo

Fazendo um comit pra ficar logo na interface do git hub

  1. Primeiro você inicializa/git init
  2. Segundo você adiciona o arquivo que irá ser mechido/ git add
  3. Terceiro você olha o status /git status
  4. Quarto você faz o comit/ git comit -m "mensagem"
  5. Quinto você modifica o nome da master(branch principal)
  6. Sexto você chama o arquivo que está no github/ faz a ligação/ git remote add origin linkdogit
  7. Sétimo você empurra o arquivo com o push / git push -u origin main
